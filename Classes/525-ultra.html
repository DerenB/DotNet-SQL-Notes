Tables

Create Table

  • CREATE TABLE <tablename>

      • <column_name> <data_type>,

      <column_name> <data_type>

    );

  • Repeat as many times as needed
  • Create Table Example:
  • CREATE TABLE cats

      • name VARCHAR(100),

      age INT

    );

  • SHOW TABLES; - shows the created tables
  • SHOW COLUMNS FROM <tablename> - Shows the table columns
  • DESC <tablename> - Shorthand to show the table columns
  • DROP TABLE <tableName> - Deletes a table

Inserting / Adding Data to Table

  • Insert two values:
    • INSERT INTO <tablename> (variablename1, variablename2) VALUES (value1, value2)
  • Order of variables doesn't matter. Just have to match variable name to value
  • To insert multiple values, separate the groups with a comma.
  • Example:

    • INSERT INTO cats(name,age) VALUES ('Peanut', 2), ('Butter', 4);

  • Shows the values in the table:

    • SELECT * FROM <tablename>

  • Add a default value to variable:

    • <varName> <varType> DEFAULT <varVALUE> ;

  • Example:

    • name VARCHAR(20) DEFAULT 'no name provided' - String with a default value

  • Add a default value to variable and prevents NULL being entered:

    • <varName> <varType> NOT NULL DEFAULT <varVALUE> ;

  • Example - String with a default value:

    • name VARCHAR(20) NOT NULL DEFAULT 'unnamed'

Keys

Creating Keys

  • Primary Key: A Unique key for each database item, determined by the specified variable.
  • PRIMARY KEY (<varName>)
  • Example:

    • CREATE TABLE unique_cats (cat_id INT NOT NULL, age INT, PRIMARY KEY (catd_id));

  • The next ID in the system can be automatically incremented
  • AUTO_INCREMENT - Add to key
  • Example:

    • CREATE TABLE unique_cats (cat_id INT NOT NULL AUTO_INCREMENT, age INT, PRIMARY KEY (catd_id));
