Notes on SQL

Creating, Deleting, Using

MYSQL-CTL CLI; - Start the CLI

SHOW DATABASES; - Lists available databases

CREATE DATABASE <name> - General Command for creating a database

DROP <name> - Deletes a database

USE <name> - Select database to use

SELECT <name> - Select database to use

Tables

CREATE TABLE <tablename>

<column_name> <data_type>,

<column_name> <data_type>

);

Repeat as many times as needed

Create Table Example:

CREATE TABLE cats (

name VARCHAR(100),

age INT

);

SHOW TABLES; - Shows the created tables

SHOW COLUMNS FROM <tablename> - Shows the table columns

DESC <tablename> - Shorthand to show the table columns

DROP TABLE <tableName> - Deletes a table

Inserting / Adding Data to Table

INSERT INTO <tablename> (variablename1, variablename2) VALUES (value1, value2) - Insert two values

Order of variables doesn't matter. Just have to match variable name to value

To insert multiple values, separate the groups with a comma. Ex:

INSERT INTO cats(name,age) VALUES ('Peanut', 2), ('Butter', 4);

SELECT * FROM <tablename> - Shows the values in the table
