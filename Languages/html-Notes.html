Flexbox

Flexbox Creation

  • Mainly used for the spacing between items and their position within the container
  • display: flex;
    • Sets a class up as a flex container
  • flex-wrap: wrap;
    • Wraps the container as the window changes size
  • flex-direction: column;
    • Makes the flex container into a column

Justify-Content

  • (default) flex-start - left horizontal align
  • center - center horizontally
  • flex-end - right horizontal align
  • space-between - extra space inbetween items
  • space-around - even amount of space around all objects

Align-Items

  • (default) stretch - fill the most amount of space vertically
  • flex-start - align to top vertically, uses item height
  • center - centers vertically

Align-Content

  • Only used for multi-line content
  • center - how much space to put between the lines

Flex Details

  • Format: "flex: flex-grow flex-shrink flex-basis"
  • Default values are "flex: 0 1 0%"
  • flex-grow takes a single digit, makes item grow n-times faster
