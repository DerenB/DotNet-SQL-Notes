Flexbox Creation

  • Mainly used for the spacing between items and their position within the container
  • display: flex;
    • Sets a class up as a flex container
  • flex-wrap: wrap;
    • Wraps the container as the window changes size
      • fsdfsdfsd
  • flex-direction: column;
    • Makes the flex container into a column
