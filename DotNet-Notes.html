Notes on DotNet

Folder Items:

Dependencies - contains the framework and packages that are required for the application to run.

Pages - contains all the pages required for the project. Has two files:

.cshtml - contains razor syntax, which is a combination of HTML markup and C#

.cshtml.cs - contains the logic and behaivor for each page

Properties - settings and definitions that are going to be used when running the app

wwwroot - contains all of the static files. Store HTML, CSS, JS, files here.

Cannot store C# or Razor files here.

Razor uses bootstrap, stored in the /lib directory.

appsettings.json - loads appsettings.json in production, loads appsettings.developer.json in development

program.cs - contains the entry point for the program. The Main Method is the first method which gets invoked.

startup.cs - contains the startup class.
